# Smart Eco Vending Machine (SEVM): Trash for Cash

## 📌 Overview

The **Smart Eco Vending Machine (SEVM): Trash for Cash** is an IoT-based smart trash bin that detects garbage automatically and releases a coin as a reward.  

The system is designed to encourage proper waste disposal by providing immediate incentives. Instead of relying solely on awareness campaigns, SEVM applies automation and embedded systems to reinforce positive environmental behavior.

---

## 🎯 Purpose

- Encourage proper waste disposal.
- Reduce littering in schools and communities.
- Promote environmental awareness.
- Motivate individuals by providing tangible rewards.
- Apply IoT technology to solve real-world environmental problems.

---

## ⚙️ How It Works

1. A user inserts trash into the bin.
2. The **Infrared (IR) sensor** detects the object.
3. The **Arduino** processes the sensor input.
4. The system:
   - Activates a **buzzer** for confirmation.
   - Triggers the **coin release mechanism**.
   - Updates the **7-segment display** (optional counter/status).

The entire process is automatic and requires no manual supervision.

---

## ✨ Features

- **Automatic Trash Detection**
  - Real-time object detection using an IR sensor.
  
- **Automatic Coin Release**
  - Coin is dispensed once trash is detected.
  
- **Audio Feedback**
  - Buzzer confirms successful trash insertion.
  
- **Visual Feedback**
  - 7-segment display shows count or system status.
